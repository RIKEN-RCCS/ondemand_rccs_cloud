5. Storage

  • R-CCS Cloud storage consists of a "Home Directory" available to each user and a "Group Directory" shared within each team/unit.
  • The Home Directory capacity is up to 1TB. To check usage: lfs quota -h -u `whoami` /hs/work0.
  • The Group Directory capacity is 5TB by default. To check usage: lfs quota -hp [Project ID] /lvs0 (e.g., lfs quota -hp 11020 /lvs0).
  • Group Directory capacity can be increased step by step (5TB → 10TB → 50TB → 100TB → 200TB → 400TB) upon request. You cannot skip steps. Submit the next request only after the previous increase has been approved.
  • Up to 400TB will be approved unconditionally. Requests exceeding 400TB will be reviewed by a committee based on the reason provided.
  • The maximum number of i-nodes in the Group Directory is 10,000,000 by default, but can be increased unconditionally upon request.
  • To apply for an increase, obtain approval from your supervisor and submit the request via this link.
