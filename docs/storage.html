4. Storage

  • The R-CCS Cloud storage provides a "home directory" available to each user and a "group directory" that can be shared within each team or unit.
  • The path to the home directory is /home/users/(username).
    • The maximum capacity of the home directory is 1 TB.
    • There is no limit on the number of files.
    • You can check usage with the following command: lfs quota -h -u `whoami` /hs/work0.
  • The path to the group directory is /lvs0/(group_name). Please refer to the table below for the group name.
    • The default maximum capacity and maximum number of files for a group directory are 5 TB and 10,000,000 files, respectively.
    • You can check usage with the following command: lfs quota -hp [Project ID] /lvs0 (example: lfs quota -hp 11020 /lvs0). Please refer to the table below for the Project ID.
    • The capacity of a group directory can be increased stepwise (5 TB → 10 TB → 50 TB → 100 TB → 200 TB → 400 TB). Skipping levels is not allowed. After your requested increase is approved, please submit the next request. Increases up to 400 TB are automatically approved. Requests exceeding 400 TB will be reviewed based on the justification provided in the application.
    • The maximum number of files can be increased upon request without conditions.
    • If you wish to apply for an increase in capacity or maximum number of files, please obtain approval from your supervisor and submit your request here.
