1. Introduction

To use the R-CCS Cloud, there are two methods: "Using Open OnDemand via a web browser" and "Connecting to the login node via SSH from terminal software".

1.1. Using Open OnDemand

Please log in at https://ondemand.cloud.r-ccs.riken.jp. Open OnDemand is a web portal that allows you to use the supercomputer from your web browser. It provides the following functions:

  • File transfer and editing
  • Terminal access
  • Execution and management of interactive applications (such as remote desktop)
  • Execution and management of batch jobs

Open OnDemand supports major web browsers such as Google Chrome, Mozilla Firefox, and Microsoft Edge (Note: Internet Explorer 11 is not supported). Chrome is recommended because it natively supports copy & paste in remote desktop and other applications.

1.2. Using Terminal Software

To register your SSH public key, please start the "SSH Public Key" application in https://ondemand.cloud.r-ccs.riken.jp. Enter your public key in the "Add Key" input field.

Next, in your terminal software, type ssh xxx@login.cloud.r-ccs.riken.jp. Here, xxx is your username.
