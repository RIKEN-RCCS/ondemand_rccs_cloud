5. ストレージ

  • R-CCSクラウドのストレージは、各ユーザが利用できる「ホームディレクトリ」と各チーム・ユニット内で共同利用できる「グループディレクトリ」があります。
  • ホームディレクトリの容量は最大1TBで、利用状況を取得するコマンドはlfs quota -h -u `whoami` /hs/work0です。
  • グループディレクトリの容量はデフォルトは5TBで、利用状況を取得するコマンドはlfs quota -hp [Project ID] /lvs0です（例：lfs quota -hp 11020 /lvs0）。
  • グループディレクトリは申請により段階的（5TB → 10TB → 50TB → 100TB → 200TB → 400TB）に増量可能です。段階を飛ばすことはできません。希望する増量が承認されてから、次の申請を提出ください。
  • 400TBまでは無条件で承認されますが、それを超えるものについては、申請理由に基づき部会で審議します。
  • グループディレクトリのi-node最大数はデフォルトで10,000,000個ですが、申請により無条件で増量できます。
  • 増量を申請する方は、所属長の許可を得てからコチラから申請ください。
